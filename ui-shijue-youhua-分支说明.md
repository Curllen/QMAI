# ui-shijue-youhua 分支说明

## 分支用途

本分支用于 QMaiWrite 软件的视觉优化与东方美学配色落地工作。

## 使用要求

- 本分支只处理与视觉、配色、UI 样式相关的改动。
- 不要在本分支中修改业务逻辑、AI 模型调用、文件读写等无关功能。
- 每次配色调整后必须运行源码验证，并打包便携版供测试。
- 合并回 main 前必须确认旧功能（章节、大纲、设置等）无回退。
